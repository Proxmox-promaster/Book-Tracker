# Book Tracker
**Автор:** Влад Чигодайкин
**Вариант:** Book Tracker (Трекер прочитанных книг)

## Описание программы
Приложение с графическим интерфейсом (GUI) для управления списком прочитанных книг. Оно помогает систематизировать библиотеку, отслеживать прогресс чтения и быстро находить нужные произведения с помощью фильтров.

### Основные возможности:
- **Учёт данных:** сохранение названий, авторов, жанров и количества страниц.
- **Валидация:** проверка на пустые поля и корректность числового ввода (страницы).
- **Фильтрация:** поиск книг по конкретному жанру или объёму (например, более 200 страниц).
- **Постоянное хранение:** все данные автоматически записываются в файл `books.json` и загружаются при старте.

## Инструкция по запуску

### Что установить:
1. Установите **Python версии 3.7** или выше с официального сайта [python.org](https://python.org).
2. Дополнительные библиотеки не требуются, так как используется встроенный модуль `tkinter`.

### Как запустить:
1. Скачайте файл с кодом (например, `main.py`).
2. Откройте терминал или командную строку в папке с этим файлом.
3. Введите команду:
   ```bash
   python main.py
   ```

## Примеры использования

### Тест 1: Добавление книги
1. Введите в поле **Название**: *Мастер и Маргарита*.
2. Введите в поле **Автор**: *Михаил Булгаков*.
3. Введите в поле **Жанр**: *Классика*.
4. Введите в поле **Страниц**: *480*.
5. Нажмите кнопку **"Добавить"**. Книга появится в общей таблице.

### Тест 2: Фильтрация по страницам
1. Введите в поле **Мин. страниц** число *300*.
2. Нажмите **"Ок"**.
3. В таблице останутся только те книги, у которых количество страниц 300 или больше.
4. Нажмите **"Сброс"**, чтобы вернуть весь список.

### Тест 3: Проверка ошибок
1. Попробуйте нажать **"Добавить"** с пустыми полями. Программа выдаст предупреждение: *"Заполните всё"*.
2. Введите в поле страниц текст (например, *"много"*). Программа выдаст ошибку: *"Страницы — число"*.
